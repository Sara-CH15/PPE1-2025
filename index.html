Mini-projet PPE1

Analyse d'URLs · Thème Robots

À propos

Ce projet collecte et analyse automatiquement des pages web sur le thème des robots, en extrayant des métadonnées essentielles pour chaque URL étudiée.

Données collectées

  • Code HTTP : Statut de la page
  • Encodage : Format de caractères
  • Nombre de mots : Volume textuel

Outils

  • Bash · cURL · Lynx
  • HTML · CSS · Bulma
  • Git · GitHub Pages
