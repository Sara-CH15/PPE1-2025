Mini-projet PPE1

Analyse automatique d'URLs sur le thème des robots

Présentation du projet

Objectif

Ce mini-projet a pour but de collecter et analyser automatiquement des pages web à partir d'une liste d'URLs sur le thème des robots.

Métadonnées collectées

Pour chaque URL, nous récupérons :

  • Code HTTP : statut de la réponse (200, 404, etc.)
  • Encodage : charset de la page (UTF-8, etc.)
  • Nombre de mots : contenu textuel de la page

Outils utilisés

  • Bash : langage de script
  • cURL : récupération des en-têtes HTTP
  • Lynx : extraction du contenu textuel
  • HTML/CSS : présentation des résultats
  • Bulma CSS : framework de style
